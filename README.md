# Script Stash

A collection of shell scripts for remote development workflows.

## Scripts

| Script | Description | Link |
|--------|-------------|------|
| `tmux-connect.sh` | Interactive tmux session manager with attach, create, rename, and retry on invalid input. | [View](tmux-connect.sh) |

## Usage

Clone the repo and run any script directly:

```bash
git clone https://github.com/hwansu93/script_stash.git
cd script_stash
chmod +x tmux-connect.sh
./tmux-connect.sh
```
